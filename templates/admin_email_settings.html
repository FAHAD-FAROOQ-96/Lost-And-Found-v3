Email Settings

Configure Gmail SMTP so the system can send real email notifications when a student ID card is found. Uses your Gmail App Password.

📋 How to set up Gmail

  1. Go to myaccount.google.com → Security
  2. Enable 2-Step Verification if not already on
  3. Search for "App Passwords" in Google Account
  4. Create a new App Password → select Mail → Other (custom name)
  5. Enter "Lost and Found" as the name → click Generate
  6. Copy the 16-character password and paste it below
